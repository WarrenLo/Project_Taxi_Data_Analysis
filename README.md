	日期x1+時間x2+氣溫x3+降雨量x4+相對濕度x5+氣壓x6+風速x7<br/>
	+海拔x8+商店x(9~?) (尚未取得)<br/>

	每區再細切10* 10(100格)，每一小格預測人數以區間顏色深淺顯示，即可顯示乘載人數預測範圍<br/>
